那么如果我们说，这个$cost$函数针对$\forall a < b < c < d$，满足：

$cost(a, d) + cost(b, c) \geq cost(a, c) + cost(b, d)$

则称这个这个函数满足四边形不等式。

当然我们根据上面那个式子可以推出来一个完全一样的等价条件，即该函数满足四边形不等式当且仅当对于$\forall a<a+1<b<b+1$，满足如下条件：

$cost(a, b + 1) + cost(a + 1, b) \geq cost(a, b) + cost(a + 1, b + 1)$。
